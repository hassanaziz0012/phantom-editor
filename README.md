Phantom Editor is a big collection of automation scripts (both deterministic and LLM-based) that helped me streamline my content creation work across YouTube, Twitter, (and in the future) any other platforms that I expand to.

This project is still heavily a work in progress, so I'll write the full README and explain each automation once this is actually finished :)

In the meantime, if you are curious, feel free to explore the codebase as you see fit.
